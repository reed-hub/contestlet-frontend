## 🔧 Development Environment

### API Server
The Contestlet API should be running at:
- **Development:** `http://localhost:8000`
- **Documentation:** `http://localhost:8000/docs`

### Frontend Integration
Use any modern web framework:
- ⚛️ React
- 🟢 Vue.js
- 🅰️ Angular
- 📦 Vanilla JavaScript
